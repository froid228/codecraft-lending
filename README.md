# CodeCraft — IT-аутсорсинг для стартапов
## Тема
IT-аутсорсинговая компания для стартапов и scale-up’ов.
## Использованные технологии
- **CSS-фреймворк**: [Pure.css](https://purecss.io/)
- **Шрифт**: Inter (Google Fonts)
- **Подход**: минимализм, компонентная структура, семантическая вёрстка
- **Адаптивность**: mobile-first, сетка Pure.css
